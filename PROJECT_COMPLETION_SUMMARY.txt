## 📊 Database Structure

### **Categories Table**
- id, name, slug ← SEO, description, image, is_active, timestamps
- 5 sample categories included

### **Products Table**
- id, category_id, category_slug ← For mapping
- name, slug ← SEO, description
- original_price, discount_price, discount_percentage
- main_image, gallery_image_1-3
- sku, stock_quantity, is_featured
- is_active, timestamps

### **Product_Variants Table**
- id, product_id ← FK
- weight_label (250g/500g/1kg)
- weight_grams, weight_value
- variant_price, variant_discount_price
- variant_sku, stock_quantity
- is_active, timestamps
